# jwp-shopping-cart

## 미션 요구사항

- [x] 상품 목록 페이지 연동 ([GET] /products)
- [x] 관리자 도구 페이지 연동 ([GET] /admin)
- [x] 상품 관리 CRUD API 작성
    - [x] 상품 생성
        - [POST] /admin/product
    - [x] 상품 목록 조회
        - [GET] /admin/products
    - [x] 상품 수정
        - [PUT] /admin/products/{id}
    - [x] 상품 삭제
        - [DELETE] /admin/products/{id}
